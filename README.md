# 🏠 Embedded Digital Assistant

A voice-controlled Digital Assistant built on embedded hardware, designed to perform essential home automation tasks such as monitoring temperature and humidity.

## 📖 About the Project

This project is an **Embedded Digital Assistant** that leverages voice recognition to interact with users and perform basic smart home functions. It is designed to run on a compact and energy-efficient platform, making it ideal for home automation setups.

The assistant can:
- Understand voice commands  
- Monitor environmental conditions  
- Provide real-time feedback  

---

## ✨ Features

- 🎤 Voice command recognition  
- 🌡️ Temperature monitoring  
- 💧 Humidity monitoring  
- 🔊 Audio feedback responses  
- ⚡ Lightweight and efficient embedded system  

---

## 🧰 Hardware

The system is built using:

- **Raspberry Pi 5** (main processing unit)  
- Temperature & Humidity Sensor
- Microphone
- Speaker
- LCD
